# Wonderland - Travel Destination Website

A beautiful, responsive travel destination website featuring nature videos and stunning visuals. Built with HTML, Tailwind CSS, and JavaScript.

## Features

- **Responsive Design**: Works on all device sizes
- **Nature Video Background**: Immersive video backgrounds
- **Dark Theme**: Elegant dark color scheme with orange accents
- **Interactive Elements**: Smooth animations and transitions
- **Multi-language**: Available in English

## Sections

- Home: Hero section with video background
- Destinations: Featured travel locations
- Facilities: Available services and amenities
- Gallery: Photo and video collection
- Testimonials: Visitor experiences
- Contact: Contact information and form

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome Icons

## Setup

1. Clone the repository
2. Open `wisata-tailwind.html` in your browser
3. Enjoy the travel experience!
